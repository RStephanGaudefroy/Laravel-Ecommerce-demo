Laravel E-commerce démo :

Simple démonstration application e-commerce comprenant pour l'instant : 
  - système d'authentification avec profil utilisateur ( 1->administrateur 2->client)
  - vue détaillant tous les achats
  - gestion du panier
  - système de paiement version test STRIPE -> Nécessite de s'inscrire pour obtenir les clés test
  - gestion simple des article type CRUD réservé à l'administrateur 
  - page contact
  - export de la base de données -> db692882236.sql

Les tests pour la responsive ont été éffectué avec Screenfly sur quirktools.com ainsi que les outils de développement de Google Chrome et Opéra et firefox Nightly.
